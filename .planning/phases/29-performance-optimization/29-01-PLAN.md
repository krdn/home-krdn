# Plan 29-01: Memoization & Context Optimization

## Goal

불필요한 리렌더링 방지를 위한 메모이제이션 및 컨텍스트 최적화

## Tasks

### Task 1: ToastProvider Context Value 메모이제이션

**파일**: `src/components/providers/ToastProvider.tsx`
**문제**: Provider value 객체가 매 렌더링마다 새로 생성
**해결**: useMemo로 context value 메모이제이션

### Task 2: AdminDashboard 위젯 메모이제이션

**파일**: `src/components/admin/AdminDashboard.tsx`
**문제**: 위젯 컴포넌트들이 매번 재렌더링
**해결**: 위젯 컴포넌트 React.memo 적용, renderWidgets useMemo 적용

### Task 3: AlertStore 정렬 로직 메모이제이션

**파일**: `src/stores/alertStore.ts` 및 관련 컴포넌트
**문제**: 알림 정렬이 매번 수행됨
**해결**: selector에서 정렬된 결과 캐싱

### Task 4: MetricsCharts 내부 차트 메모이제이션

**파일**: `src/components/admin/MetricsCharts.tsx`
**문제**: 개별 차트 컴포넌트가 memo로 감싸지지 않음
**해결**: MetricsLineChart, NetworkAreaChart를 memo로 감싸기

- [ ] 기존 테스트 통과
