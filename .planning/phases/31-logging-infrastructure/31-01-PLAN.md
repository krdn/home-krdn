# Plan 31-01: Logger 서비스 구현

## Overview

pino 기반 중앙집중식 로깅 인프라를 구축합니다.

## Tasks

### Task 1: pino 설치 ✅

```bash
npm install pino pino-pretty
```

### Task 2: Logger 서비스 구현 ✅

**파일**: `src/lib/logger.ts`

**기능**:
- 환경별 설정 (개발: pretty-print, 프로덕션: JSON)
- 로그 레벨: trace, debug, info, warn, error, fatal
- 컨텍스트 바인딩 (child logger)
- 요청별 로거 (createRequestLogger)
- 모듈별 로거 (createModuleLogger)

### Task 3: error-logger.ts 마이그레이션 ✅

기존 console 기반 로깅을 pino logger로 교체:
- logError: 에러 레벨 자동 분류 (4xx: warn, 5xx: error)
- logWarning, logInfo, logDebug 헬퍼 추가
