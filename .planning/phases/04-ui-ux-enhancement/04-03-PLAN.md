Card 컴포넌트 호버 효과 개선:


   ```tsx

   "hover:shadow-lg hover:border-primary/20 hover:-translate-y-0.5"
   ```

3. **그룹 호버 지원:**
   ```tsx
   // group 클래스 추가하여 자식 요소도 반응 가능

     "group rounded-lg border bg-card text-card-foreground shadow-sm",
     "transition-all duration-200",
     hover && "cursor-pointer hover:shadow-lg hover:border-primary/20 hover:-translate-y-0.5",
     className
   )}
   ```

4. **CardContent 내부 아이콘 호버 효과 (선택적):**

기존 hover prop 유지하되 효과 강화.
