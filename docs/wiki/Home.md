## 빠른 시작

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build

# Docker 배포
docker-compose up -d
```

## 주요 기능

### 홍보 기능
- 메인 랜딩 페이지
- 서비스 카탈로그 (카테고리별 필터링)
- 개별 서비스 상세 페이지

### 관리 기능
- 대시보드 (시스템 개요)
- 서비스 시작/중지
- Docker 컨테이너 관리
- 실시간 로그 조회
- 시스템 리소스 모니터링
